## Placeholder-сервисы:

Для демо-версии можно использовать следующие сервисы для получения placeholder-изображений:

1. **Unsplash** (реальные фотографии спорта):
   - https://unsplash.com/s/photos/sports
   - https://unsplash.com/s/photos/basketball
   - https://unsplash.com/s/photos/football

2. **Picsum** (рандомные изображения):
   - https://picsum.photos/400/300

3. **Placehold** (простые placeholder'ы):
   - https://placehold.co/400x300

## Рекомендации для реального сайта:

1. Используйте профессиональные фотографии студентов во время тренировок
2. Фотографии должны быть высокого разрешения (минимум 1920x1080 для hero-изображений)
3. Обязательно получите согласие на использование изображений людей
4. Оптимизируйте изображения для веба (сжатие, правильные форматы)
5. Добавьте альтернативный текст для всех изображений

## Оптимизация изображений:

- Формат WebP для современных браузеров
- JPEG для фотографий с потерями
- PNG для логотипов и графики с прозрачностью
- Размер файлов не должен превышать 500KB для веб-изображений
